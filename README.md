## 📦 What art is

art is a local developer tool for the SmartWiz+ Art Digital Frame.

It helps you work with your frame from a Windows PC. You can use it to set up the frame, send content, and test how it looks on an ePaper display.

This project is built for home automation and smart home use. It fits well if you want a simple way to manage a digital frame without cloud tools.

## 💻 What you need

Before you start, check these items:

- A Windows 10 or Windows 11 PC
- An internet connection for the download
- A SmartWiz+ Art Digital Frame
- A USB cable or network setup, if your frame uses one
- Permission to run apps on your PC

## 🛠️ First use

When you open art for the first time, connect your SmartWiz+ Art Digital Frame.

If the app asks for a device path, frame address, or access setting, use the value that matches your frame setup.

If your frame uses the local network, make sure your PC and frame are on the same Wi-Fi network.

If your frame uses USB, make sure the cable is firmly connected.

## 🔌 Common connection tips

If the frame does not show up in the app:

- Check the USB cable
- Restart the app
- Restart the frame
- Make sure Windows did not block the app
- Confirm that your PC and frame are on the same network
- Try a different USB port

If the app finds the frame but the screen stays blank:

- Check the image size
- Check the display mode
- Try a new test image
- Reconnect the device

## 🧰 Troubleshooting

### The download will not open

- Download the file again
- Right-click the file and choose Open
- Check that the file finished downloading
- Try a different browser

### Windows shows a security prompt

- Choose Run only if the file came from the Releases page
- If the file is blocked, right-click it and check Properties
- Try downloading the latest release again

### The frame is not detected

- Unplug and plug in the cable
- Close and reopen art
- Check Wi-Fi on both devices
- Try another USB port
- Restart the frame

### The display looks wrong

- Use a larger image
- Remove extra text
- Make the contrast stronger
- Match the frame resolution
- Try a test image with only black and white shapes
